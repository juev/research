# Research

Коллекция аналитических обзоров по инструментам и технологиям.

Опубликовано на <https://research.evsyukov.org/>

## Статьи

- **Сравнительный анализ ИИ моделей (март 2026)** — Claude Opus 4.6, MiniMax M2.5/M2.7, Kimi K2.5 и другие: стоимость, бенчмарки, практическая эффективность
- **Cursor IDE как LLM-proxy** — аналитический обзор использования Cursor IDE в качестве прокси к LLM
- **Утилита LLM от Simon Willison** — аналитический обзор CLI-инструмента для работы с языковыми моделями
- **Управление буфером обмена через SSH** — подходы к проброске clipboard при работе на удалённых серверах

## Локальная разработка

Для предпросмотра сайта локально:

```bash
pip install -r requirements.txt
mkdocs serve
```

Сайт будет доступен по адресу <http://127.0.0.1:8000/>

## Деплой

Сайт автоматически собирается и публикуется на GitHub Pages при push в ветку `main` через GitHub Actions.

## Технологии

- [MkDocs](https://www.mkdocs.org/) — генератор статических сайтов
- [Material for MkDocs](https://squidfundamentals.com/mkdocs-material/) — тема оформления
